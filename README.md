# proyecto-studio-mochi
Crear una aplicación en base a Flutter con Dart con soporte de plataforma para administrar una CRUD de un estudio fotográfico con todas las plataformas android/web/windows
CRUD para aplicación estudio fotográfico profesional y moderna llamada 'Studio mochi 22px'.
(analiza todos los archivos como sea necesario).
Diseño moderno, profesional y limpio pero no sencillo, utiliza una paleta de colores basada en: #D4CFC3 #384057 #B3B3B3 #E8E8E6 #2E3B6E  .
tu pon un logo del negocio .
La aplicación debe poder ser visible correctamente y sin problemas en un dispositivo móvil
 y el layout deberá estar basado para administrar
la aplicación inicia con un login funcional (iniciar sesión, registrarse, olvidó su contraseña) 
Incluye un banner superior azul con bordes inferiores redondeados que incluya el logo y un mensaje de bienvenida. Incluye un footer que diga 'Herrera Sanchez Valeria. 2026 6°I' como derechos de autor.
Y después de hacer eso dar acceso al crud de fotógrafos y sesiones de fotos , firebase y firestore, en una pantalla de administración que muestre botones con icono y con el nombre de cada tabla, después al presionar mandar a la tabla correspondiente de cada crud
un crud de fotógrafos  (agregar, listar, actualizar y borrar), con los campos Certificado o prueba de un titulo Profesional(string),seccion (string),Horario de Turno(string), Nombre(string), Salario(double).
un crud sesiones de fotos (agregar, listar, actualizar y borrar), con los campos Bajo Prescripción(boolean), Estudio(string), Nombre Comercial(string), Presentación(string), Stock(int64)
utilizar widgets atractivos y modernos, usar firebase autenticación, Cloud Firestore, no se te olvide integrar las dependencias y configuraciones necesarias actualizar pubspec.yaml, mi proyecto es console firebase es: studio-mochi22px, integración de los archivos .dart necesarios en la carpeta bin y subcarpetas, proyecto totalmente funcional
